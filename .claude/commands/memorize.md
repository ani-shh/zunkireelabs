# MEMORIZE SESSION - Zunkiree Website

You are **Zunkiree-Stark** completing a work session.

## MEMORIZE PROTOCOL

Save this session's learnings to the project brain.

---

## Step 1: Gather Session Information

Before creating the session log, collect:

1. **What was accomplished?** (features, fixes, decisions)
2. **What patterns were discovered?** (reusable learnings)
3. **What decisions were made?** (and why)
4. **What blockers exist?** (if any)
5. **What's the next logical step?**

---

## Step 2: Create Session Log

Create a new file in `.brain/session-logs/` with format:
`YYYY-MM-DD-brief-description.md`

### Session Log Template:

```markdown
# Session Log: [Brief Description]

**Date:** [Today's date]
**Duration:** [Approximate time]
**Session Type:** [Development/Planning/Bug Fix/etc.]

---

## What Was Accomplished

- [Bullet points of completed work]

---

## Decisions Made

### [Decision Title]
- **Decision:** [What was decided]
- **Reasoning:** [Why]
- **Impact:** [What it affects]

---

## Patterns Discovered

### [Pattern Name] (if any)
- **Pattern:** [Description]
- **When to Use:** [Context]
- **Example:** [Code or workflow example]

---

## Blockers/Issues

- [Any blockers encountered]
- [Or "None"]

---

## Next Steps

1. [Next logical action]
2. [Following action]

---

## Files Changed

- [List of significant files modified]

---

## Notes for Future Sessions

- [Context that future sessions should know]
```

---

## Step 3: Update Status File

Update `.brain/status-current.md` with:
- New "Last Updated" date
- Current phase/sprint status
- Recent activity
- Updated blockers
- Next steps

---

## Step 4: Update Decisions Log (if applicable)

If significant decisions were made, add to `.brain/decisions-log.md`

---

## Step 5: Consider Global Patterns

If a pattern was discovered that applies across projects:
- Note it for `/memorize-global` in the main brain
- Flag it in the session log as "Cross-Project Pattern"

---

## Step 6: Confirm Save

After completing all updates, confirm:

```
SESSION MEMORIZED

Saved:
- Session log: .brain/session-logs/[filename]
- Status updated: .brain/status-current.md
- [Decisions log updated: Yes/No]

Key Learnings:
- [1-2 main takeaways]

Next Session:
- [What to pick up next time]

Ready for: git commit or continue working
```
